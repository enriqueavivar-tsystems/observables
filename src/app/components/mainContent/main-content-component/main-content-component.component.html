Uso básico y términos

Los observables brindan soporte para pasar mensajes entre editores y suscriptores en su aplicación. Los observables ofrecen beneficios significativos sobre otras técnicas para el manejo de eventos, la programación asincrónica y el manejo de múltiples valores.

Como editor, crea una Observable instance que define una subscriber function . Esta es la función que se ejecuta cuando un consumidor llama al método subscribe(). La función subscriber define cómo obtener o generar valores o mensajes para publicar.

Para ejecutar el observable que ha creado y comenzar a recibir notificaciones, llame a su método subscribe(), pasando un observador . Este es un objeto de JavaScript que define los controladores para las notificaciones que recibe. La llamada a subscribe() devuelve un objeto Subscription que tiene un método unsubscribe(), al que llama para dejar de recibir notificaciones.

Definición

Un controlador para recibir notificaciones observables implementa la interface del Observer. Es un objeto que define métodos de devolución de llamada para manejar los tres tipos de notificaciones que un observable puede enviar:

Subscribing

Una instancia Observable comienza a publicar valores solo cuando alguien se suscribe a ella. Se suscribe llamando al método subscribe() de la instancia, pasando un objeto observador para recibir las notificaciones.

Creación de Observables

Use el constructor Observable para crear una secuencia observable de cualquier tipo. El constructor toma como argumento la función de suscriptor que se ejecuta al ejecutarse el método subscribe () del observable. Una función de suscriptor recibe un objeto Observador y puede publicar valores en el método next () del observador.

Multidifusion

Un observable típico crea una nueva ejecución independiente para cada observador suscrito. Cuando un observador se suscribe, el observable conecta un controlador de eventos y entrega valores a ese observador. Cuando un segundo observador se suscribe, el observable conecta un nuevo controlador de eventos y entrega valores a ese segundo observador en una ejecución separada.
