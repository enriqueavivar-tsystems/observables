Creación de Observables

Use el constructor Observable para crear una secuencia observable de cualquier tipo. El constructor toma como argumento la función de suscriptor que se ejecuta al ejecutar-se el método subscribe () del observable. Una función de suscriptor recibe un objeto Observador y puede publicar valores en el método next () del observador.

Multidifusion

Un observable típico crea una nueva ejecución independiente para cada observador suscrito. Cuando un observador se suscribe, el observable conecta un controlador de eventos y entrega valores a ese observador. Cuando un segundo observador se suscribe, el observable conecta un nuevo controlador de eventos y entrega valores a ese segundo observador en una ejecución separada.

A veces, en lugar de comenzar una ejecución independiente para cada suscriptor, desea que cada suscripción obtenga los mismos valores, incluso si los valores ya han comenzado a emitirse. Este podría ser el caso con algo así como un observable de clics en el objeto del documento.

La multidifusión es la práctica de transmitir a una lista de múltiples suscriptores en una sola ejecución. Con un observable de multidifusión, no registra varios oyentes en el documento, sino que reutiliza el primer oyente y envía valores a cada suscriptor.

Al crear un observable, debe determinar cómo desea que se use ese observable y si desea o no multidifusión de sus valores.
